# 文件解析助手 - 第一阶段 MVP 实现计划

> **版本**: MVP v1.0  
> **日期**: 2026-03-09  
> **阶段**: 第一阶段（最小可行产品）

---

## 概述

第一阶段只实现 MVP 核心功能，确保项目可行性。不做完整的企业级功能（多用户、策略市场等）。

---

## 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | Next.js 15+ (App Router) + React 19+ |
| **UI** | Tailwind CSS + shadcn/ui |
| **后端** | Node.js + Express |
| **数据库** | SQLite（先简单，后续迁移到 MySQL） |
| **AI 服务** | SiliconFlow API + MinerU API |
| **风格** | 蓝白简约 + 液态玻璃 (Glassmorphism) |

---

## 任务清单

### 任务 1: 项目初始化与基础架构
- [ ] 初始化 Next.js + Express 项目结构
- [ ] 配置 Tailwind CSS + shadcn/ui
- [ ] 创建基础页面布局（侧边栏 + 主内容区）
- [ ] 配置液态玻璃 (Glassmorphism) 样式

### 任务 2: 文件上传与管理
- [ ] 文件上传组件（支持多格式）
- [ ] 文件列表展示（侧边栏）
- [ ] 文件删除功能
- [ ] 文件保存（本地存储）

### 任务 3: PDF 解析 - 模式 A（默认）
- [ ] 集成 pdfplumber
- [ ] 直接文本提取
- [ ] 表格提取
- [ ] 扫描件检测（文本 < 阈值自动触发 OCR）

### 任务 4: PDF 解析 - 模式 B（MinerU）
- [ ] 集成 MinerU API
- [ ] MinerU Key 配置
- [ ] 调用 MinerU 进行深度解析
- [ ] Markdown/JSON 结果展示

### 任务 5: 其他文件格式解析
- [ ] Word (.docx) 解析
- [ ] Excel (.xlsx/.xls) + CSV 解析
- [ ] TXT/JSON 直接读取
- [ ] 统一的解析结果展示

### 任务 6: OCR 识别
- [ ] 集成 PaddleOCR-VL（SiliconFlow）
- [ ] 图片/扫描件 OCR
- [ ] 分片处理（大图片）

### 任务 7: 策略管理
- [ ] 系统预设策略列表
- [ ] 策略编辑器（创建/编辑/删除）
- [ ] 策略选择器（解析时选择）
- [ ] 策略存储（localStorage）

### 任务 8: AI 对话界面
- [ ] Chat 界面（流式输出）
- [ ] 基于文件内容的 AI 对话

### 任务 9: 设置页面
- [ ] 模型配置（主模型/OCR/ASR）

### 任务 10: UI 完善与优化
